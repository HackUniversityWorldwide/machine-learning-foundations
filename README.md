A beginner-friendly workshop series designed for students who want to start learning machine learning from zero, with simple hands-on activities and an easy learning path.

## About this series

This repository contains materials for the **Machine Learning Foundations** workshop series.  
The goal is to make machine learning approachable, practical, and fun for beginners.

This series is part of **Build with AI Study Jams** and is organized by **Google Developer Group at FIU**.

## Workshop series overview

### Week 1 — Getting Started with Google Colab
Introduction to Google Colab, notebooks, code cells, text cells, simple Python practice, and a first mini hands-on activity.
