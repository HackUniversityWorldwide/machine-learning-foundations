# Machine Learning Foundations

**Build with AI Study Jams | C1**  
**Hosted by Google Developer Group at FIU**  
**Worldwide | All Majors | All Universities**

A beginner-friendly workshop series for students who want to start learning machine learning from zero in a simple, practical, and fun way.

## About the series

This repository contains materials for the **Machine Learning Foundations** workshop series.

The goal is to help beginners get comfortable with the basics of machine learning by learning step by step, starting with tools like **Google Colab**, simple Python, and small hands-on activities.

This series is designed to feel approachable:
- no prior machine learning experience required
- beginner-friendly workshop style
- short, guided activities
- learn while building
